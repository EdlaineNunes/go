#### Instalação do Golang

#### Linux: 
- Gerenciador de pacotes:
    ~$ sudo snap install --classic go
              *ou*
    ~$ sudo apt-get install golang-go
   
 - Imagem Docker: 
     ~$ docker pull golang
              *ou*
     ~$ docker pull golang:1.16.5
